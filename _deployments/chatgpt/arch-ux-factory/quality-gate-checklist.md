# Quality Gate Checklist: Transition Standards

Before a handoff is approved by the Bmad Master, these checks must PASS:

### Gate 1: Portfolio -> Vision

- [ ] Strategic intent is clear.
- [ ] Budget/Resources allocated (simulated).

### Gate 2: Vision -> Architecture

- [ ] PRD has North Star metric.
- [ ] User Personas defined.
- [ ] Epics have WSJF scores.

### Gate 3: Architecture -> Delivery

- [ ] ADR explains the 'Why' for major tech choices.
- [ ] Infrastructure diagram is present.
- [ ] Tech Spec Bridge contains API/Database impacts.

### Gate 4: Delivery -> Quality

- [ ] User Stories have Gherkin ACs.
- [ ] Rally/Jira CSV mapping is complete.

### Gate 5: Quality -> Master (Final Release)

- [ ] Traceability Matrix shows 100% coverage.
- [ ] Adversarial Review completed with 0 Critical blockers.
